Tips y reglas para conexión a clase por Zoom 
  
.

 Tips para conexión a clase por Zoom
 
  • Para acceder a la aplicación de ZOOM, entra al siguiente URL https://itesm.zoom.us/
  • Te aparecerá una página WEB con varias opciones. Selecciona la opción JOIN.

                   

  • El ID de la sesión es el siguiente: 4641421192

                        

  • En caso de NO tener instalado el cliente de ZOOM en tu computadora, sigue las instrucciones para descargar y ejecutar el cliente de ZOOM en tu computadora. Este proceso tardará cerca de dos minutos.
  • Ejecuta el archivo que se ha descargado para instalar el cliente de ZOOM.

                           

  • Utiliza tu nombre y apellido durante la sesión.  



 Reglas para las sesiones virtuales síncronas

  • Al darse de alta en zoom deben usar su nombre y apellido.

  • Tener siempre la cámara encendida.

  • Asegurarse de tener batería suficiente o tener su computadora conectada de modo que no se les corte la sesión por esta razón.

  • Cerrar las aplicaciones de su computadora que no están usando para la clase.

  • Abrir los documentos que se estén usando para la sesión de clase.

  • Usar audífonos y micrófonos.

  • Estar solos en una habitación.
